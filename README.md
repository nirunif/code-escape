# Code Escape

Code Escape is a puzzle-style mini-game designed to challenge players with five sequential levels. Each level must be cleared to unlock the next, encouraging progression, strategy, and careful thinking.

## Game Overview

- The game consists of 5 levels, each with its own challenge.
- Players must complete Level 1 â†’ Level 5 in order.
- A level unlocks only after the previous one is completed.

## Attempts & Reset Mechanic

Each level gives the player 3 attempts. If a player fails all three attempts on any level, the game resets back to Level 1. This mechanic adds tension, replayability, and encourages mastery.

## Key Features

- Structured progression through five stages
- Limited attempts per level
- Automatic reset on full attempt depletion
- Simple logic-based gameplay loop

## How to Play

- Start at Level 1.
- Complete the level to unlock the next.
- Avoid losing all three attempts at any level.
- Reach and clear Level 5 to win the game!
